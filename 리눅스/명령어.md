단축키
	Tab : 명령자동완성
	C+c : 중단
	C+a : 맨 앞으로 커서 이동
	C+e : 맨 뒤로 커서 이동
	C+r : history 검색
	
파일시스템 관련 명령어
	cd(change directory)
	cd - : 바로 직전 경로로 이동
	ls(list)
	ls -al
	ls -1
	ls -alh
	df(disk free)
디렉토리 명령어
	mkdir
	rmdir 
	pwd(print working directory)
파일통계 명령어
	mount : 디스크장치 표시/디렉토리 연결
	umount : mount 해제
	stat : 파일 통계 출력
파일관련 명령어
	touch : 빈 파일 생성 / 타임스탬프 업데이트
	cat(catenate)
	head
	tail
	cp -rfp
		-r : 하위 디렉토리 포함
		-f : 강제
		-p : 권한 포함
	mv
	rename
	rm -rf
	less
